# Chicago Crime Prediction using Big Data and Machine Learning

This project analyses large-scale crime data using machine learning and big data technologies.

## Dataset
Chicago Crimes – 2001 to Present  
Source: Chicago Open Data Portal

https://data.cityofchicago.org/Public-Safety/Crimes-2001-to-Present/ijzp-q8t2

The dataset contains millions of crime records including crime type, location, time and arrest status.

## Technologies Used
Python  
PySpark  
Apache Spark MLlib  
Machine Learning  
Tableau  

## Machine Learning Models
Logistic Regression  
Decision Tree  
Random Forest  
Gradient Boosted Trees
